
DUMMY PERSONAL ASSISTANT (1.0)
-------------------------------------


-------------------------------------------------------------------
In this we simply search from wikipedia and wolfram alpha using speech recognition and display the result in a tkinter window
so before writing the code sign up in https://www.wolframalpha.com/  
and sign in 
then follow  the following clicks
-------------------------------------------
/Account(your gmail)/My Apps(API)/getapp Id
------------------------------------------------------------------ 
this app Id is used in the code which authorizes us to use their api for search requests 
limit is 2000 per day
---------------------------------------------------------------------------
while running in your system please make sure following modules are installed
type the following commands in terminal to install the required modules

pip install SpeechRecognition
pip install wolframalpha
pip install wikipedia
pip install tkinter
---------------------------------------------------------------------------
sometimes an error occurs like : "AttributeError: Could not find PyAudio; check installation"
then type the following commands in the terminal
-------------------------
pip install pipwin
pipwin install pyaudio
----------------------------------
then run the code again 
your Dummy Personal Assistant without AI is ready.
------------------------------------------------------
Also use a headphone or earphone mic for best results as the ordinary code sometimes doesnt support inbuilt system mics
